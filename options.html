Proxmox Quick Control

Connect to your Proxmox VE cluster using an API token.

Connection

If port is omitted, 8006 is used. Visit the URL once in your browser to accept the TLS certificate before saving.
Format: user@realm!tokenname
Create in Proxmox: Datacenter → Permissions → API Tokens → Add. For full control, give the token role PVEAdmin (or a narrower role with VM/CT/Node power permissions). Uncheck "Privilege Separation" if the user already has the needed permissions.

Behavior

Wake on LAN

Chrome extensions can't send UDP magic packets directly. WoL uses Proxmox's /nodes/{node}/wakeonlan endpoint, so it requires another node in the cluster to be online and the offline node's MAC address configured under Datacenter → Node → Edit → Wake-on-LAN.
